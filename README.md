# Simon Game

A web-based Simon Game implementation. Challenge your memory by repeating the sequence of colors displayed by the game. Each level increases the length of the sequence, testing your ability to recall longer chains.

## Features

- Classic Simon Game experience
- Visually engaging UI with animated buttons
- Game logic implemented in JavaScript
- Responsive design for desktop and mobile devices

## How to Play

1. Press the **Start** button to begin.
2. Watch the game play a sequence of button flashes and sounds.
3. Repeat the sequence by clicking the colored buttons in order.
4. The game will add a new color to the sequence each round.
5. Make a mistake, and the game ends. Try again to beat your previous score!
